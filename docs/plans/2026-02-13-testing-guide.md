## 数据库验证

### 验证项目列表查询包含contractId

```sql
-- 执行项目列表查询，检查是否返回contractId
SELECT
    p.project_id,
    p.project_name,
    pcr.contract_id as contractId
FROM pm_project p
LEFT JOIN pm_project_contract_rel pcr ON p.project_id = pcr.project_id
LIMIT 10;
```

**预期结果**:
- 查询成功执行
- 有合同的项目返回对应的contract_id
- 无合同的项目contractId为NULL

**验证点**:
- [ ] SQL执行无错误
- [ ] LEFT JOIN正确关联
- [ ] contractId字段存在

---

### 验证合同唯一性约束

```sql
-- 查询项目合同关联表
SELECT
    project_id,
    contract_id,
    COUNT(*) as count
FROM pm_project_contract_rel
GROUP BY project_id, contract_id
HAVING COUNT(*) > 1;
```

**预期结果**:
- 查询结果为空（没有重复关联）

**验证点**:
- [ ] 无重复关联记录
